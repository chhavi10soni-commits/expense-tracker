# Expense Tracker

A simple and interactive Expense Tracker Web App built using HTML, CSS, and JavaScript.
It helps users track daily expenses, categorize spending, and visualize data using charts.

--

## Features

• Add expenses with title, amount, and category
• Categories: Food, Travel, Shopping, Other
• View total money spent
• Expense history with delete option
• Data stored in browser using LocalStorage
• Pie chart visualization of spending

## Tech Used

• HTML
• CSS
• JavaScript
• Chart.js

## How It Works

• User enters expense details
• Data is saved in browser storage
• Total amount updates automatically
• Chart shows category-wise spending
• User can delete any entry

## How to Run

• Download or clone this repository
• Open `tracker.html` in any browser
• Start adding your expenses
